I.     內容:[r], [ l ], 與音節化[ ļ ], [ ], [ņ]

1.子音

[r]：舌頭向上捲起，舌尖不可碰觸硬顎，振動聲帶發出捲舌音。此音在母音前發音似國語的ㄖ，在母音後時則音似國語的ㄦ。

   [ l ]：雙唇微開，舌尖抵住上牙齦，振動聲帶，氣息自兩邊流出。此音在母音前類似國語的ㄌ。

2. [ ļ ], [ ], [ņ]為音節化的音。

[ ļ ]：先發[ə]的音，再發[ l ]的音，是[əl]的結合。

  [ ]：先發[ə]的音，再發[m]的音，是[əm]的結合。

  [ņ]：先發[ə]的音，再發[n]的音，是[ən]的結合。

III.           發音規則：請注意！發音規則是依大部份狀況來歸類，但非絕對的鐵則，僅供參考。

1. [r]：此音是由字母r所發的。

2. [ l ]：此音是由字母l所發的。
